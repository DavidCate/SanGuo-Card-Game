当前游戏版本：1.0
未命名 在线
未命名 0 50
    本游戏为回合制游戏，玩家初始血量值为30，灵活的卡牌对战会对玩家的血量产生不同的效果，当某一方玩家的血量值先降为0，则失败，另一方获得胜利；多人游戏中只有一人可获胜，即活到最后一人为胜利方。
    本游戏分为匹配模式和故事模式。
    匹配模式：玩家可在线与一个或多个玩家（最多七人）进行同台竞技卡牌战斗，活到最后即为胜利。
    故事模式：玩家可逐一选择一项历史事件关卡来进行卡牌战斗，战斗胜利即为闯关成功，失败则可以重来。每闯过一关可开启下一关卡，每关都有丰厚的战斗奖励等你来拿。
    卡牌对战规则：卡牌可分为帝王牌，武将牌，军师牌，武器牌，城池牌，玉玺牌。帝王牌与武将牌代表不同的攻击力与攻击效果；军师牌代表不同的计谋与战斗效果；武器牌代表对玩家的攻击力加持效果；城池牌代表对玩家的防御力加持效果；玉玺牌代表对玩家的血量值回复效果。
    恭喜你已成功加入三国群英榜，可尽情与其他英雄豪杰一决高下，争夺榜首之位。     不删档1.0版本上线，全部英雄卡牌免费使用。
